限量: 单次返回所有历史数据

|-----|-----|-----|
| -   | -   | -   |

|-----------|--------|-----|
| 序号        | int64  | -   |
| 营业部名称     | object | -   |
| 携手营业部家数   | int64  | -   |

| 年内最佳携手股票数 | int64  | -   |
| 年内最佳携手成功率 | object | -   |

```python
import akshare as ak
```
